# Mpox Genomics Analysis Pipeline

This project shows my Mpox bioinformatics workflow from raw FASTQ files to consensus genome generation.

## Workflow

1. Quality control using FastQC
2. Read trimming using fastp
3. Mapping to reference genome using BWA
4. BAM processing using SAMtools
5. Variant calling using FreeBayes
6. Consensus sequence generation using bcftools

## Tools Used

- FastQC
- fastp
- BWA
- SAMtools
- FreeBayes
- bcftools

## Skills Demonstrated

- Linux command line
- FASTQ file handling
- Genome alignment
- Variant calling
- Consensus FASTA generation
- Viral genomics workflow
